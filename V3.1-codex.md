# V3.1-codex 작업 파일

## 목표

- 사용자에게 바로 보이는 버전명을 `V3.1-codex`로 명확히 표기
- 6개 키트의 `준비물 확인`과 `부품 확인 미션` 문구를 공통 규칙으로 통일
- `STEP-BY-STEP 교육 과정` 진행 패널을 스크롤 고정으로 유지
- 대용량 키트 이미지를 경량화해 실제 초기 로딩을 개선

## 이번 반영 범위

- README에 `V3.1-codex` 진입 링크와 최근 변경 요약 추가
- `v3.1-codex.html` 신규 생성
- 기존 `v3-codex.html`은 호환용 리다이렉트로 유지
- 공용 스크립트에서 버전 배지, 푸터, 타이틀을 variant query 기준으로 동기화
- 공용 학습 섹션에서 `준비물 확인`/`부품 확인 미션` 문구와 안내 문장을 통일
- `assets/images/kits` 대용량 이미지를 리사이즈/재압축해 페이지 진입 속도 개선

## 사용자에게 보이는 핵심 차이

- 상단 버전 표기가 `V3.1-codex`로 표시됨
- README에서 현재 codex 작업 버전이 바로 식별됨
- 6개 키트 모두 같은 표현으로 `준비물 확인`과 `부품 확인 미션`이 안내됨
- 교육 과정 진행도가 스크롤 중에도 계속 보임
- 첫 화면과 키트 사진 로딩이 이전보다 가벼워짐
