# DORO Science Voucher Education Portal

`DORO` 과학문화바우처용 교육 포털 저장소입니다.  
학생과 보호자가 키트를 선택하면 구성품 확인, 안전 약속, 만들기, 기록, 자료 확인까지 한 화면에서 따라갈 수 있도록 만든 버전형 가이드입니다.

## 버전 변화 요약

| 구간 | 변화 |
|---|---|
| V1 → V2 | 소개형 초기안에서 실제 키트 선택형 포털 구조로 전환 |
| V2 → V3 | 학생/보호자가 바로 쓰기 쉬운 운영형 메인 포털로 정리 |
| V3 → V4 | 학습 동선을 더 명확하게 나누고 QR·구성품·안전 단계 강화 |
| V4 → V5 | 세계관과 픽셀 UI를 붙여 브랜드 톤과 몰입감을 강화 |
| V5 → V6 | 자산, 문서, 영상, 데이터 파이프라인을 정리해 완성도를 높임 |
| V6 → V7 | 사용자 피드백 기준으로 선택 카드, 체크 완료 상태, CTA, 문구, 접근성을 재정비 |
