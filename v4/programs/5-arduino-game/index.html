START CHECK

시작 전에 이것만 확인해요

1. 작업 자리

작은 부품과 점퍼선이 섞이지 않게 책상 위를 먼저 정리해요.

2. 전원 상태

전선을 꽂기 전에는 보드 전원이 연결되지 않았는지 먼저 확인해요.

3. 도움 요청

핀 번호나 화면 방향이 헷갈리면 혼자 억지로 꽂지 말고 바로 도움을 요청해요.

BUILD GUIDE

한 단계씩 만들기

안전 약속을 끝내면 아래 만들기 흐름을 따라갈 수 있어요.

안전 약속을 먼저 모두 확인해 주세요.

안전 section에서 체크를 모두 마치면 만들기 section이 열려요.

1
왜 이 키트를 배우는가⏱ 15분

게임기는 어떤 과학 원리로 작동하는지, 입력-출력 구조를 소개합니다.

  • 게임기의 구성: 입력(버튼) + 처리(코드) + 출력(화면/소리)
  • 아두이노란? 피지컬 컴퓨팅의 기본
  • 완성 게임 시연
2
오늘의 과학 개념⏱ 10분

입력장치와 출력장치의 종류, 디지털 신호의 개념을 학습합니다.

  • 입력: 버튼, 센서, 조이스틱
  • 출력: LED, 화면, 부저
  • 디지털 신호: 0과 1의 세계
  • 활동지 입력-출력 분류 카드
3
조립 + 작동 확인⏱ 45분

하드웨어를 조립하고, 선 연결과 전원 상태를 차례대로 확인합니다.

  • OLED + 버튼 + 부저 브레드보드 연결
  • 보드와 화면 자리 확인
  • 배터리와 전원 연결 확인
  • 케이스 조립
4
게임 플레이 + 탐구⏱ 20분

내가 만든 게임기로 직접 게임을 플레이하고, 작동 원리를 분석합니다.

  • 러닝 점프 게임 플레이
  • 버튼 입력 → 화면 반응 관찰
  • 점수 시스템 작동 원리 분석
  • 활동지 게임 동작 분석 카드
5
정리 + 발표⏱ 30분

피지컬 컴퓨팅 개념을 정리하고, 내 게임기를 소개합니다.

  • 입력-처리-출력 흐름 정리
  • 내 게임기 시연 발표
  • 확장 미션: 나만의 게임 아이디어 설계
TROUBLESHOOTING

왜 안 되지?

OLED의 SDA, SCL 핀이 아두이노의 A4, A5에 정확히 연결되었는지 확인하세요. USB 케이블을 통해 전원이 공급되어야 합니다.
버튼의 네 다리가 브레드보드에 올바르게 꽂혀 있는지 확인하세요. 점퍼선 연결 핀 번호가 코드와 일치해야 합니다.
전원을 끄고 5초 뒤 다시 켜 보세요. 그래도 화면이 이상하면 학생이 다시 설정하려 하지 말고 보호자나 선생님께 보여 주세요.
