📖 STEP-BY-STEP 교육 과정

1 도입 — 게임기의 원리 ⏱ 10분
  • 완성된 파란 돼지 게임기 시연
  • 입력(버튼) → 처리(아두이노) → 출력(LCD) 구조 소개
  • 피지컬 컴퓨팅 개념 소개
2 과학·코딩 개념 ⏱ 15분
  • 아두이노: 마이크로컨트롤러 기반 오픈소스 보드
  • 디지털 입력: 버튼 누름 = 0 또는 1
  • LCD: 문자와 숫자를 표시하는 출력 장치
  • 게임 로직: 조건문과 반복문으로 구현
3 조립 — 게임기 제작 ⏱ 35분
  • 파란 돼지 목재 케이스 조립
  • 아두이노 UNO 장착
  • LCD 디스플레이 연결 (I2C 방식)
  • 점프 버튼 장착 및 연결
  • 배터리 홀더 연결
4 코딩 — 러닝 점프 게임 만들기 ⏱ 30분

제공된 예시 코드를 수정하며 나만의 게임을 만듭니다.

  • USB로 아두이노와 노트북 연결
  • 기본 코드 업로드 → 게임 작동 확인
  • 장애물 속도 변경 / 점수 규칙 수정
  • 코드 수정 → 업로드 → 테스트 반복
5 게임 대회 + 발표 ⏱ 30분
  • 친구와 점수 경쟁
  • 내가 수정한 규칙 발표
  • 피지컬 컴퓨팅 개념 정리

❓ 자주 묻는 질문

LCD 뒷면의 I2C 주소(보통 0x27 또는 0x3F)를 코드에서 확인하세요. 밝기 조절 나사를 돌려 대비를 조정해볼 수도 있습니다.
버튼의 핀 연결 방향을 확인하고, 코드의 digitalRead 핀 번호가 실제 연결 핀과 일치하는지 확인합니다.
네. 제공된 기본 코드를 업로드하면 즉시 게임이 작동합니다. 수치 값만 바꿔도 게임이 달라지므로 코딩 경험이 없어도 창작 활동이 가능합니다.
arduino.cc에서 Arduino IDE 2.x를 설치하고, USB 케이블로 연결 후 「도구 → 포트」에서 아두이노가 연결된 COM 포트를 선택합니다. Windows에서 포트가 보이지 않으면 CH340 USB 드라이버를 별도 설치하세요. 수업용 코드 파일은 교사 지도서에 포함되어 있습니다.
아두이노 보드에 5V 이상의 전압을 직접 연결하지 마세요. 핀 번호를 잘못 연결하면 부품이 손상될 수 있으니 회로도를 꼼꼼히 확인합니다. USB 연결 상태에서 도선이 단락(쇼트)되지 않도록 주의하고, 작업 전 정전기 방지를 위해 손을 닦아주세요.
중학교 정보 교과 「프로그래밍」·「피지컬 컴퓨팅」 영역과 직결됩니다. 변수, 조건문, 반복문 등 알고리즘 기초 개념을 게임 코드로 직접 체험하며, 초등 실과 「소프트웨어와 생활」 단원의 확장 활동으로도 적합합니다.
