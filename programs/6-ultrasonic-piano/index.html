🎹 도짜르트 — 초음파 피아노

초음파 센서와 조건문으로 거리별 음 높이를 제어하는 인터랙티브 피아노를 만들며, 사운드 인터랙션과 피지컬 컴퓨팅을 탐구합니다.

📖 STEP-BY-STEP 교육 과정

1
왜 이 키트를 배우는가⏱ 10분

초음파 센서가 어떻게 거리를 재는지, 소리와 주파수의 관계를 소개합니다.

  • 초음파란? 사람이 들을 수 없는 높은 소리
  • 초음파 → 반사 → 시간 차이 = 거리
  • 피아노 건반과 주파수의 관계
  • 완성된 피아노 시연
2
오늘의 과학 개념⏱ 10분

초음파 거리 측정의 수학적 원리와 조건문의 개념을 학습합니다.

  • 거리 = 속도 × 시간 / 2
  • 조건문: if (거리 < 5cm) → 도, else if (거리 < 10cm) → 레…
  • 주파수(Hz)와 음높이의 관계
  • 활동지 음계-거리 매핑 카드 작성
3
조립 + 코딩⏱ 30분

초음파 센서, 부저, LED를 연결하고 코드를 업로드합니다.

  • 브레드보드에 초음파 센서 연결 (Trig/Echo)
  • 피에조 부저 연결
  • LED 8개 음계별 연결
  • 피아노 코드 업로드 (QR 코드 → 다운로드)
  • 케이스 프레임 조립
4
연주 + 실험⏱ 25분

손의 거리를 조절하며 피아노를 연주하고, 음계 실험을 합니다.

  • 손을 가까이/멀리 하며 음 높이 변화 확인
  • 도레미파솔라시도 연주 시도
  • LED가 음계에 맞춰 점등되는지 관찰
  • '학교 종이 땡땡땡' 같은 간단한 곡 도전
  • 활동지 실험 기록지 작성
5
정리 + 발표⏱ 15분

초음파 센서와 사운드 인터랙션 원리를 정리합니다.

  • 초음파 거리 측정 핵심 정리
  • 조건문과 음계 매핑 복습
  • 확장 미션: 도로랜드 '하모니아의 노래' 연주
선택형 확장 활동

🎵 AI 도구로 나만의 곡 만들기

기본 키트만으로도 학습이 완결됩니다. 추가로 Suno나 ChatGPT를 활용하면 AI가 생성한 멜로디를 초음파 피아노로 연주하거나, 나만의 작곡 스토리를 만드는 창작 활동이 가능합니다.

※ AI 도구 미사용 시에도 기존 곡 연주와 음계 실험만으로 학습 목표를 달성합니다.

❓ 자주 묻는 질문

피에조 부저의 +/- 극성과 연결 핀 번호를 확인하세요. 부저가 제대로 꽂혀 있는지 브레드보드 확인도 필요합니다.
초음파 센서 앞에 장애물이 없는지 확인하세요. 센서와 손 사이에 3~30cm 범위가 가장 정확합니다. 코드의 거리 구간 값을 조절해 볼 수도 있습니다.
각 LED의 연결 핀 번호가 코드에 지정된 핀과 일치하는지 확인하세요. LED 극성(긴 다리가 +)과 저항 연결도 점검합니다.
