📖 STEP-BY-STEP 교육 과정

1 도입 — 손이 악기가 되다 ⏱ 10분
  • 완성된 개구리 피아노 시연 (손 거리로 음 조절)
  • 초음파가 거리를 측정하는 원리 소개
  • 테레민이나 에어 기타 같은 비접촉 악기 사례
2 과학·코딩 개념 ⏱ 15분
  • 초음파 센서: 음파 발사 → 반사 → 시간으로 거리 계산
  • 소리의 높낮이: 진동수(Hz)에 따라 결정
  • 조건문: 거리 구간별 다른 음 재생
  • 활동지 음계 매핑 표 작성
3 조립 — 개구리 피아노 제작 ⏱ 20분
  • 개구리 피아노 건반 보드 조립
  • 아두이노 UNO 장착
  • 초음파 센서 연결
  • 피에조 부저 + LED 연결
  • LCD 디스플레이 연결
4 코딩 — 거리-음 매핑 ⏱ 25분

조건문으로 손의 거리에 따라 다른 음이 나오도록 코딩합니다.

  • 기본 코드 업로드 → 손으로 거리 조절 테스트
  • if-else 조건문으로 음계 구간 설정
  • tone() 함수로 피에조 부저 주파수 제어
  • LCD에 현재 음 이름 표시하기
5 연주 + 발표 ⏱ 20분
  • 간단한 멜로디 연주 도전
  • 친구와 합주 시도
  • 사운드 인터랙션 개념 정리

❓ 자주 묻는 질문

초음파 센서의 Trig/Echo 핀 연결을 확인하고, 피에조 부저 핀 번호가 코드와 일치하는지 확인하세요.
초음파 센서 정면에 장애물이 고정되어 있지는 않은지 확인하세요. 센서 앞 30cm 이내에 아무것도 없는 공간이 필요합니다.
코드의 if 조건문에서 각 음의 거리 구간을 더 넓게 설정하면 원하는 음을 더 안정적으로 낼 수 있습니다.
arduino.cc에서 Arduino IDE 2.x를 설치 후 USB로 연결하고, 「도구 → 포트」에서 아두이노 COM 포트를 선택합니다. Windows에서 포트가 보이지 않으면 CH340 USB 드라이버를 별도 설치하세요. NewPing 라이브러리가 필요한 경우 「도구 → 라이브러리 관리」에서 설치합니다.
코드에서 tone() 함수의 주파수 값을 낮추거나, 피에조 부저 위에 작은 스티커를 붙이면 소리가 부드러워집니다. 수업 환경에 따라 음량을 미리 조절해 두는 것을 권장합니다.
초등·중학교 과학 「파동과 소리」 영역, 중학교 정보 교과 「피지컬 컴퓨팅」 단원과 연계됩니다. 초음파 거리 측정과 주파수 매핑은 수학 교과의 함수·비례 개념과도 자연스럽게 연결됩니다.
