# 나노바나나 Pro — DORO 픽셀 아이콘 의뢰 프롬프트

> **의뢰 방법**: 아이콘마다 [공통 스타일] + [개별 프롬프트]를 함께 입력하세요.  
> 총 **24개** 아이콘 (교체 우선 높음 기준).

---

## 공통 스타일 지시문 (모든 아이콘에 반드시 포함)

```
Minecraft Java Edition 16x16 pixel art icon texture style.
Exactly 16x16 pixels. Transparent PNG background.
Dark navy outline color #0b1528 (very dark navy-black, NOT pure black).
3-tone flat shading: dark face, base color, 1-2 highlight pixels.
No anti-aliasing, no gradients, no smooth edges — every pixel must be sharp and crisp.
Child-friendly warm color palette. Simple iconic silhouette readable at 16px size.
Game UI icon aesthetic — bold, chunky pixels, instantly recognizable shape.
Style reference: Minecraft game texture pack + classic 16-bit JRPG UI icons.
```

## 출력 요청 사항

- **파일 형식**: PNG, 투명 배경
- **크기**: 16×16 px (또는 32×32 px 업스케일)
- **저장명**: `[키이름].png` (예: `build.png`, `science.png`)
- **색상**: 팔레트 최대 8색 이하 per 아이콘
- **참고**: 마인크래프트 텍스처팩 스타일, 포켓몬스터 도트 아이콘 스타일
