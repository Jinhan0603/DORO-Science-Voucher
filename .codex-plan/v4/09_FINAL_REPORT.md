# Final Report Instructions

When all phases are complete, produce a final report with:

1. Solved feedback list
2. Unsolved feedback list
3. Modified file list
4. New asset list
5. 6 kit status table:
   - PDF
   - PPT-derived images
   - assembly video
   - demo video
   - component images
   - build steps
   - troubleshooting
   - safety gate
6. Large file check result
7. Desktop test result
8. Mobile test result
9. Known risks
10. Recommended next human review points

Also run:

```bash
git status
git diff --stat
find assets -type f -size +90M -print
```
